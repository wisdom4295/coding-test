## 오늘 공부 총정리

### 📌 핵심 개념

- `Math.trunc()` — 소수점 버림, 0 방향으로 절사. 음수에서도 안전
- `~~` 비트 연산자 — `Math.trunc()`와 동일하지만 양수에서만 안전
- `Array.reverse()` — **원본 배열 직접 변경(mutate)** 주의. 불변이 필요하면 `[...arr].reverse()` 또는 `toReversed()` 사용
- `Array.from(String(n), Number)` — 숫자를 자릿수 배열로 변환하는 핵심 패턴
- `reduce` 초기값 — 빈 배열 가능성이 있으면 항상 초기값 `0` 명시

### 💡 새로 익힌 패턴

- 문자열 인덱싱: `'abcdefghij'[숫자]` — 객체 매핑 없이 간결하게 문자 변환
- `String.fromCharCode(n + 97)` — ASCII 코드로 a~j 매핑
- 짝수의 합 수학 공식: `k = Math.floor(n/2)` → `k * (k + 1)` — O(1) 풀이
- 화살표 함수 `{}` 유무에 따른 반환 차이:
  - `(x) => x + 1` — 자동 반환
  - `(x) => { return x + 1; }` — 명시적 return 필수

### ❌ 아쉬운 점

- 짝수의 합: 배열을 만들어 filter/reduce로 풀었으나, 수학 공식(O(1))으로 더 효율적으로 풀 수 있었음
- 외계행성의 나이: 객체 매핑 대신 문자열 인덱싱 패턴을 미리 알았다면 코드가 더 간결했을 것
