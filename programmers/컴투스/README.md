## 📌 합격 전략 핵심

### 시간 분배 원칙
- **CS 50% : JS/React 30% : 알고리즘 20%**
- 컴투스 후기상 코딩 일부 못 풀어도 합격 가능
- 서술형·객관식이 합격의 키

### 문제풀이 5-15-10 룰
| 단계 | 시간 | 할 일 |
|------|------|-------|
| 분석 | 5분 | 입출력·제약조건·예시 손으로 적기 |
| 풀이 | 15분 | 종이에 의사코드 → 코드 작성 |
| 학습 | 5분 | 막히면 해답 본 후 즉시 손으로 다시 작성 |

> ⚠️ **한 문제 25분 절대 사수.** 1시간 매달리면 다른 7문제 못 풉니다.

### 강의 시청 원칙
- 절대 처음부터 다 보지 않기
- 문제를 먼저 풀고 → 그 문제 강의만 발췌
- 새 강의 시작 금지, **진도 60% 이상 강의만 활용**

## 🎯 시험 당일 시간 배분 전략

1. **첫 5분** — 전체 문제 스캔, 어려운 문제 표시
2. **객관식 먼저** — 빠르게 결정 가능한 점수부터 확보
3. **서술형** — 구조 잡고 작성 (서론 1줄 → 본론 2~3개 → 마무리)
4. **코딩** — 마지막에. 부분 점수 노리기, 작동하는 코드부터 제출
5. **빈칸 두지 않기** — 객관식은 찍기, 서술형은 아는 만큼이라도
