# 외계행성의 나이

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120834)
- 레벨: Lv.0
- 풀이 시간: 15분

***

## 내 풀이

```javascript
const alphabetObj = {
    0: 'a', 1: 'b', 2: 'c', 3: 'd', 4: 'e',
    5: 'f', 6: 'g', 7: 'h', 8: 'i', 9: 'j'
};

function solution(age) {
    const tempArr = Array.from(String(age), Number);
    const replaceArr = tempArr.map(item => alphabetObj[item]);
    return replaceArr.join('');
}
```

## ✅ 잘한 점

- 매핑 테이블을 객체로 명시적으로 정의 → 가독성 높음
- `Array.from` → `map` → `join` 흐름이 명확

## ⚠️ 아쉬운 점 / 주의사항

- 객체 대신 문자열 인덱싱으로 더 간결하게 표현 가능 (베스트 풀이 참고)
- `Number`로 변환 후 다시 객체 키로 쓰는 과정이 한 단계 더 있음

## 💡 베스트 풀이

```javascript
// 문자열을 배열처럼 인덱싱 활용
const solution = (age) =>
    age.toString()
       .split('')
       .map(v => 'abcdefghij'[v])
       .join('');

// String.fromCharCode 활용
const solution = (age) =>
    [...String(age)].map(n => String.fromCharCode(+n + 97)).join('');
```

## 📌 핵심 개념

- `'abcdefghij'[v]`: 문자열도 배열처럼 인덱스 접근 가능
- `String.fromCharCode(n + 97)`: a=97, b=98 ... ASCII 코드 활용
- 객체 매핑 vs 문자열 인덱싱 — 길이 고정된 경우 문자열 인덱싱이 간결
