# 두 수의 나눗셈

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120806)
- 레벨: Lv.0
- 풀이 시간: 5분

***

## 내 풀이

```javascript
function solution(num1, num2) {
    return Math.trunc((num1 / num2) * 1000);
}
```

## ✅ 잘한 점

- `Math.trunc()`를 활용해 소수점 이하 버림을 정확하게 처리
- 곱셈 먼저 연산해 정밀도 확보

## ⚠️ 아쉬운 점 / 주의사항

- `Math.floor()`와의 차이 주의: `Math.floor(-1.5) = -2`, `Math.trunc(-1.5) = -1`
- 음수가 포함될 경우 `Math.trunc`가 올바른 선택

## 💡 베스트 풀이

```javascript
// ~~ 비트 연산자 활용 (양수에서만 동작)
function solution(num1, num2) {
    return ~~((num1 / num2) * 1000);
}
```

## 📌 핵심 개념

| 방법 | 설명 | 음수 처리 |
|------|------|-----------|
| `Math.trunc(x)` | 소수점 이하 버림 (0 방향) | ✅ 안전 |
| `Math.floor(x)` | 내림 (음의 무한대 방향) | ⚠️ 음수 주의 |
| `~~x` | 비트 NOT 2회 = trunc와 동일 | ⚠️ 양수만 안전 |
| `parseInt(x)` | 문자열 파싱 후 정수 | ⚠️ 성능 느림 |
