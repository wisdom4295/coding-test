# 두 수의 합

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120802)
- 레벨: Lv.0
- 풀이 시간: 5분

***

## 내 풀이

```javascript
function solution(num1, num2) {
    return num1 + num2;
}
```

## ✅ 잘한 점

- 가장 간결한 형태로 정답 작성
- 불필요한 변수 없이 바로 반환

## ⚠️ 아쉬운 점 / 주의사항

- 이 문제의 **심화 버전**은 num1, num2가 **문자열(string)**로 주어져 일반 `+` 연산이 불가
- 그 경우 `BigInt`를 사용해야 함: `return String(BigInt(num1) + BigInt(num2))`

## 💡 베스트 풀이

```javascript
// 동일 — 이 문제는 단순 덧셈이므로 내 풀이가 베스트
function solution(num1, num2) {
    return num1 + num2;
}
```

## 📌 핵심 개념

- `+` 연산자: 숫자면 덧셈, 문자열이면 연결(concatenation)
- 입력 타입을 항상 먼저 확인할 것 (number vs string)
