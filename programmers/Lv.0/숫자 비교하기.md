# 숫자 비교하기

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120807)
- 레벨: Lv.0
- 풀이 시간: 5분

***

## 내 풀이

```javascript
function solution(num1, num2) {
    if(num1 === num2) return 1;
    return -1;
}
```

## ✅ 잘한 점

- `===` 엄격 비교 사용 (타입까지 비교)
- 간결하게 조기 반환

## ⚠️ 아쉬운 점 / 주의사항

- `==` 대신 `===` 사용 습관은 좋음 — 유지할 것

## 💡 베스트 풀이

```javascript
// 삼항 연산자로 한 줄 처리
const solution = (num1, num2) => num1 === num2 ? 1 : -1;
```

## 📌 핵심 개념

- `===` (엄격 동등): 값 + 타입 모두 비교
- `==` (느슨한 동등): 타입 변환 후 비교 → 코테에서 지양
- 삼항 연산자: `조건 ? 참 : 거짓`
