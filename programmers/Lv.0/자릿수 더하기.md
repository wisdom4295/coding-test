# 자릿수 더하기

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120906)
- 레벨: Lv.0
- 풀이 시간: 15분

***

## 내 풀이

```javascript
function solution(n) {
    const tempArr = Array.from(String(n), Number);
    return tempArr.reduce((acc, curr) => acc + curr);
}
```

## ✅ 잘한 점

- `Array.from(String(n), Number)`: 숫자를 문자열로 변환 후 각 자릿수를 숫자 배열로 변환하는 깔끔한 패턴
- 두 번째 인자 매핑 함수를 활용한 점이 좋음

## ⚠️ 아쉬운 점 / 주의사항

- `reduce`에서 초기값 생략 → 첫 번째 요소가 acc 시작값
- 이 문제에선 문제 없지만, 빈 배열 가능성 있는 경우 초기값 `0` 명시 권장

## 💡 베스트 풀이

```javascript
// split + map + reduce
const solution = (n) =>
    String(n).split('').reduce((acc, cur) => acc + Number(cur), 0);
```

## 📌 핵심 개념

- `Array.from(iterable, mapFn)`: 이터러블을 배열로 변환하면서 동시에 매핑
- `String(n).split('')`과 `Array.from(String(n))` 결과 동일
- 숫자 → 문자 → 숫자 배열 변환 패턴: 자릿수 관련 문제에서 자주 사용
