# 배열 뒤집기

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120821)
- 레벨: Lv.0
- 풀이 시간: 10분

***

## 내 풀이

```javascript
function solution(num_list) {
    return num_list.reverse();
}
```

## ✅ 잘한 점

- `reverse()`로 가장 간결하게 해결

## ⚠️ 아쉬운 점 / 주의사항

- `Array.reverse()`는 **원본 배열을 직접 변경(mutate)** 함
- 원본 보존이 필요한 경우 `[...num_list].reverse()` 또는 `num_list.toReversed()` 사용
- 코테에서는 원본 변경이 문제 없는 경우가 많지만 습관적으로 인지할 것

## 💡 베스트 풀이

```javascript
// 원본 불변 방식 (ES2023 toReversed)
const solution = (num_list) => num_list.toReversed();

// 또는 spread
const solution = (num_list) => [...num_list].reverse();
```

## 📌 핵심 개념

| 메서드 | 원본 변경 | 지원 |
|--------|-----------|------|
| `reverse()` | ✅ 변경 | ES5+ |
| `toReversed()` | ❌ 불변 | ES2023+ |
| `[...arr].reverse()` | ❌ 불변 | ES6+ |
