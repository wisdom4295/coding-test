# 짝수의 합

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120831)
- 레벨: Lv.0
- 풀이 시간: 10분

***

## 내 풀이

```javascript
function solution(n) {
    const tempArr = Array.from({ length: n }, (_, i) => i + 1)
        .filter(i => i % 2 === 0);

    return tempArr.reduce((acc, curr) => acc + curr, 0);
}
```

## ✅ 잘한 점

- `Array.from`으로 1~n 배열 생성 → `filter`로 짝수 추출 → `reduce`로 합산
- 함수형 체이닝 패턴을 자연스럽게 사용

## ⚠️ 아쉬운 점 / 주의사항

- 중간 배열(`tempArr`)을 만드는 게 메모리 낭비 — 체이닝으로 한 번에 처리 가능
- 세미콜론 두 번 (`;;`) 오타 주의

## 💡 베스트 풀이

```javascript
// 수학 공식 활용 — O(1) 시간복잡도
// n 이하 짝수의 합 = 2+4+...+2k = k*(k+1), k = Math.floor(n/2)
function solution(n) {
    const k = Math.floor(n / 2);
    return k * (k + 1);
}

// 또는 for문으로 직관적으로
function solution(n) {
    let answer = 0;
    for (let i = 2; i <= n; i += 2) answer += i;
    return answer;
}
```

## 📌 핵심 개념

- `Array.from({length: n}, (_, i) => i + 1)`: 1~n 배열 생성 패턴
- `filter(i => i % 2 === 0)`: 짝수 필터
- 수학 공식으로 O(1) 풀이 가능한지 항상 고려할 것
