# 배열의 평균값

- 출처: [프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/120817)
- 레벨: Lv.0
- 풀이 시간: 10분

***

## 내 풀이

```javascript
function solution(numbers) {
    let total = numbers.reduce((acc, curr) => {
        return acc + curr;
    }, 0);
    return total / numbers.length;
}
```

## ✅ 잘한 점

- `reduce`로 합산 후 길이로 나누는 정석 패턴 활용
- 초기값 `0` 명시 → 빈 배열 방어

## ⚠️ 아쉬운 점 / 주의사항

- `reduce` 콜백에서 중괄호 `{}` 사용 시 `return` 필수 — 잘 지켜짐
- 더 짧게 쓸 수 있음 (아래 베스트 풀이 참고)

## 💡 베스트 풀이

```javascript
// 화살표 함수 + 암묵적 반환으로 압축
const solution = (numbers) =>
    numbers.reduce((acc, curr) => acc + curr, 0) / numbers.length;
```

## 📌 핵심 개념

- `reduce(callback, initialValue)`: 배열 → 단일 값으로 축약
- 초기값 생략 시 첫 번째 요소가 초기값 → 빈 배열에서 에러 발생
- `{}` 있으면 명시적 `return` 필수, `{}` 없으면 자동 반환
